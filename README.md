# InstaSort

Локальное Android-приложение на Expo Router для сортировки сохраненных Instagram-постов по своим категориям.

## Что умеет

- добавлять посты по URL или вручную;
- хранить посты, категории и связи в `expo-sqlite`;
- искать, фильтровать и сортировать коллекцию;
- писать заметки к постам;
- автоматически подбирать категорию полностью офлайн;
- создавать первую категорию автоматически, если список еще пуст;
- экспортировать и импортировать базу JSON.

## Запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите приложение:

```bash
npx expo start
```

3. Для Android-эмулятора или устройства:

```bash
npx expo run:android
```

## Важно про Expo Go и SDK 54

Эта версия проекта зафиксирована на Expo SDK 54 ради совместимости с Expo Go. Основной сценарий импорта работает через вставку URL вручную.

Что это значит:

1. Откройте пост в Instagram.
2. Скопируйте ссылку.
3. Вставьте ее на экране `Add Media`.

Если позже понадобится именно системный Android share intent, для него лучше делать отдельный dev build и подключать нативное решение вместо Expo Go-режима.
